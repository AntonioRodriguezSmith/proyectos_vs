# modeloRagAtenas

Esta carpeta está reservada exclusivamente para el modelo RAG central y subproyectos temáticos relacionados con Alejandría.

- No contiene la gestión global de informes ejecutivos.
- Toda la lógica de organización y automatización de informes está documentada en el README de la raíz de informes_ejecutivos.
- Si se crea un subproyecto aquí, documenta su propósito y relación en el INDEX.md central de Alejandría.

Consulta siempre la biblioteca central antes de iniciar desarrollos en esta carpeta.

# ADR 001_informe_dxc

## Proyecto: Informe IA en DXC

### Contexto
Este ADR documenta las decisiones, aprendizajes y referencias clave del informe ejecutivo sobre la implantación de IA en DXC Technology.

### Decisiones principales
- Uso de Python, Markdown y JSON para automatización y documentación.
- Estructura modular: cada informe tiene su propia carpeta y presentación.
- Integración de métricas y visualizaciones interactivas (HTML/Chart.js).

### Conocimientos relevantes
- Automatización de tareas repetitivas y gestión de datos de prueba.
- Analítica avanzada y reporting automatizado.
- Buenas prácticas de trazabilidad y documentación.

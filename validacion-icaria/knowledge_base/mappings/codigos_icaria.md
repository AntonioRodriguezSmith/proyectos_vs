# Catálogo de Códigos ICARIA

## Tipo de Persona
| Código | Descripción |
|--------|-------------|
| ES_PERSONA_FISICA | Usuario tipo Particular |
| ES_PERSONA_JURIDICA | Usuario tipo Empresa |
| ES_AUTONOMO | Usuario Autónomo |

| Código | Descripción |
|--------|-------------|
| TIENE_BANCA_A_DISTANCIA | Contrato BD Particulares |
| TIENE_BANCA_A_DISTANCIA_EMPRESA | Contrato BD Empresas |
| TIENE_BANCA_A_DISTANCIA_AUTONOMO | Contrato BD Autónomos |

## Cuentas
| Código | Descripción |
|--------|-------------|
| TIENE_CUENTA_VISTA | Tiene cuenta a la vista |

| TIENE_MAS_DE_X_CUENTAS_VISTA_EMPRESA | Tiene más de X cuentas empresa |
| TIENE_MOVIMIENTOS_EN_CUENTA | Tiene movimientos en cuenta |

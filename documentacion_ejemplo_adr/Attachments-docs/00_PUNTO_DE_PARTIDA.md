# Resumen Ejecutivo del Proyecto: Sistema Colaborativo Data Vault + IA

## Misión Principal

El objetivo es desarrollar un **sistema colaborativo que utiliza una IA experta en Data Vault 2.0** para asistir a los equipos de datos en el modelado y la toma de decisiones arquitectónicas. El sistema debe ser una **herramienta de apoyo** que acelere el desarrollo, no un decisor automático.

## Principio Arquitectónico Clave: "Código Mínimo, IA Máxima"

La filosofía del proyecto es **delegar todo el razonamiento complejo a la IA conversacional**. El software (backend y frontend) se limita a orquestar las conversaciones, gestionar la persistencia y visualizar los resultados. No se debe reimplementar lógica que la IA ya puede gestionar.

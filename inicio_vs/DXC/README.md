# Recuerdos de configuración DXC

Objetivo: Tener un script de inicio personalizado para cada PC (DXC, ARS, MTP), usando un script base reutilizable y comprobaciones globales.

Este archivo README servirá para anotar ideas, decisiones y pasos importantes durante la creación y configuración de los scripts de inicio para cada PC.

## Notas iniciales
- El objetivo es tener un script de inicio personalizado para cada PC (DXC, ARS, MTP).
- Se usará un script base reutilizable para comprobaciones globales y utilidades.
- Cada script de inicio podrá guardar información local del equipo la primera vez que se ejecute.
- Las comprobaciones globales se ejecutarán siempre que se inicie el script.

# Historial de sesión y contexto activo

## Reglas y acuerdos recientes
- Registrar siempre un log de contexto con el estado, decisiones y temas tratados, aunque sean provisionales.
- No es obligatorio usar literalmente los nombres que propone el usuario (ej: "tronco" y "ramas"), salvo que se indique expresamente.
- Si el usuario mezcla temas o enlaza proyectos, el asistente debe organizar, clarificar y documentar el avance.
- Si el asistente no entiende algo, debe preguntar antes de asumir o ejecutar.
- Automatizar la actualización y sincronización de los proyectos que lo requieran.
- El asistente debe procesar y estructurar las ideas del usuario, salvo que se pida un formato específico.

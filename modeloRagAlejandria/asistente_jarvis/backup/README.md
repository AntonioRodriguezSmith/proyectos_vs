# Backup de recursos y documentación de modeloJarvis

Este directorio contiene copias de seguridad de los recursos clave y la documentación esencial para restaurar o consultar el funcionamiento de modeloJarvis.

## Archivos incluidos

- patrones.json: Patrones de comportamiento y respuesta
- modos_trabajo.json: Modos especiales de interacción
- diccionario.json: Diccionario de expresiones y términos personalizados
- instrucciones.md: Instrucciones de uso, automatización y troubleshooting
- mensaje_chat.txt: Mensaje de bienvenida o ejemplo para el chat

## Notas

- Estos archivos son una copia de referencia. La versión activa y centralizada está en modeloJarvis/src/conducta/recursos y la documentación en modeloJarvis/README.md y docs/.
- Actualiza este backup tras cambios importantes en los recursos o la documentación.

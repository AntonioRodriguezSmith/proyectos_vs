**Norma global:** Cada proyecto debe mantener su propio archivo de tareas local (por ejemplo, `tareas_pendientes.md`). El directorio central de tareas solo debe contener tareas globales o enlaces a los archivos de tareas locales. Si un proyecto requiere mayor trazabilidad, puede incluir un README de tareas en su propio directorio de tareas.

# modeloRagAlejandria (Biblioteca Central)

## Descripción general

Este proyecto es la biblioteca central de conocimiento para el sistema RAG (Retrieval-Augmented Generation), inspirada en la Biblioteca de Alejandría. Aquí se consolida la información, documentación, decisiones y recursos útiles para todos los proyectos relacionados.

## Repositorio en GitHub

Todos los proyectos y subproyectos relacionados con modeloRagAlejandria deben estar bajo control de versiones usando Git. El repositorio principal se encuentra en:

- [https://github.com/AntonioRodriguezSmith](https://github.com/AntonioRodriguezSmith)

Se recomienda clonar este repositorio y mantener sincronizados los cambios. Para nuevos subproyectos, crea un repositorio propio o una carpeta dentro de este, siempre bajo control de versiones.

## Convención de nombres y estructura

- Cada subproyecto llevará el nombre de una ciudad histórica (ejemplo: modeloRagAtenas, modeloRagEsparta, etc.).
- Esta carpeta principal se denomina modeloRagAlejandria y actúa como el "centro de conocimiento".
- Los proyectos pueden residir en diferentes PCs y sincronizarse mediante Git.
- Entornos de trabajo principales: DXC (este entorno), ARS y MTP. Documenta en qué entorno trabajas para facilitar la colaboración y sincronización.

# modeloRagAtenas

Esta carpeta está reservada como contenedor para futuros subproyectos, módulos o experimentos temáticos relacionados con el sistema Alejandría.

- No es un proyecto independiente por sí mismo.
- Toda coordinación, documentación y decisiones deben centralizarse en modeloRagAlejandria.
- Cuando se cree un nuevo subproyecto aquí, documenta su propósito y relación en el INDEX.md central.

Consulta siempre la biblioteca central antes de iniciar desarrollos en esta carpeta.
